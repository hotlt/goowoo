💭 为什么叫「狗窝」？

以前认真投入工作的时候，我喜欢在桌上堆放大量的资料、书籍、笔记。

同事们走进办公室，总会说："你这屋像个狗窝！"

但正是这种方式，让我在需要什么资料时能马上找到，效率极高。

领导要什么，我随手就能拿出来。别人给的资料，我随手就放到桌上。

"狗窝"不是脏乱差，而是有序的热闹 —— 是投入工作的痕迹，是效率的证明。

✨ 核心特性

🏠

本地存储

SQLite 轻量数据库，零依赖，隐私安全。你的数据只属于你。

🔍

全文检索

FTS5 全文索引，毫秒级搜索。想找什么，一呼即应。

🏷️

分类管理

给内容打标签，按分类筛选。井井有条，随存随取。

🔥

热点优先

越常用的内容排序越靠前。常用信息，一眼看到。

🔄

重复合并

相似内容自动检测，避免冗余。知识库保持精简。

💾

定时备份

覆盖式备份，数据永不丢失。安心使用，无后顾之忧。

📖 快速上手

命令 用法 说明
add add "内容" [关键词] [分类] 添加内容，自动提取关键词
search search "关键词" FTS 全文搜索，热点优先
get get <id> 获取条目完整内容
update update <id> "新内容" 更新已有内容
list list [分类] 列出所有内容
delete delete <id> 删除指定条目
stats stats 查看统计信息
backup backup 备份数据库
